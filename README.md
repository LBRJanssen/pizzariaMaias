## Sobre o Projeto

Pizzaria Maia's é um site completo desenvolvido com **React + Vite**, apresentando uma pizzaria com design moderno e responsivo. O projeto foi migrado de HTML/CSS/JS puro para React, mantendo todas as funcionalidades e melhorando significativamente a organização do código.

### Evolução do Projeto

Este projeto começou em 2022 como um site estático em HTML/CSS/JavaScript puro, demonstrando habilidades fundamentais em desenvolvimento web. Em 2024, foi completamente refatorado para React + Vite, mostrando evolução técnica e aprendizado contínuo.

## Funcionalidades

- **Página Inicial** - Apresentação da pizzaria com seções sobre nós, destaques do cardápio e características
- **Cardápio Completo** - Página dedicada com todas as pizzas disponíveis organizadas por categorias, com imagens para cada pizza
- **Contato** - Formulário de contato funcional com validação e informações de contato
- **Localização** - Mapa interativo do Google Maps e informações sobre como chegar
- **Design Responsivo** - Adaptado para todos os dispositivos (desktop, tablet e mobile)
- **Animações Suaves** - Transições e efeitos visuais modernos
- **Acessibilidade** - Código semântico e acessível com atributos ARIA

## Tecnologias Utilizadas

- **React 19.2.0** - Biblioteca JavaScript para construção de interfaces
- **Vite 7.2.4** - Build tool e dev server extremamente rápido
- **React Router DOM 7.10.1** - Roteamento para aplicações React
- **CSS3** - Estilização com animações e gradientes modernos
- **Google Fonts** - Fontes Teko, Redressed e Inter

## Como Executar

### Pré-requisitos

- Node.js (versão 18 ou superior)
- npm ou yarn

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/LBRJanssen/pizzariaMaias.git
cd pizzariaMaias
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o projeto em modo de desenvolvimento:
```bash
npm run dev
```

4. Abra o navegador em `http://localhost:5173`

### Build para Produção

```bash
npm run build
```

O build será gerado na pasta `dist/`.

### Preview do Build

```bash
npm run preview
```

## Cardápio

O cardápio inclui imagens reais de pizzas para cada item:

- **Pizzas Tradicionais**: Margherita, Pepperoni, Calabresa, Quatro Queijos
- **Pizzas Especiais**: Maia's Especial, Portuguesa, Frango com Catupiry, Bacon
- **Pizzas Doces**: Chocolate, Brigadeiro, Banana com Canela
